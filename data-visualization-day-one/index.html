Data Visualization - Day One

2015-07-13 at 11:08

Today, I focused primarily on preparing the front end structure of this project.

To start, I setup this blog to track our daily progress. It is built on the node-based blogging platform Ghost.io and hosted on Github via a static page generator, Buster.

The next step was to explore the javascript data visualization library D3 JS. I started by importing a dataset that Josh provided. This dataset shows the frequency of operating systems used in the user population from the Kaggle Competition. The biggest challenge that I encountered in visualizing the data was choosing how to best reorganize the dataset to easily build a bar chart in D3. I found that building a bar chart was easiest when each entry was formatted as a JSON object with an id and frequency.

There are a few remaining problems with the bar chart as it stands now. It is not responsive, it cuts off entries beyond 56, it lacks a color gradient, and it is not interactive. The above features are all things that I am going to implement prior to demo day. So much left to do in the days to come. Check out the bar chart below!
